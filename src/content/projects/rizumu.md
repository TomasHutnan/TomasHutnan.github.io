Rizumu is a rhythm-based game developed by the AE development team, focusing on responsive gameplay mechanics driven by music and player interaction. The project aimed to create an engaging and customizable rhythm experience with a strong emphasis on UI responsiveness and player feedback.

The gameplay revolves around synchronizing user input with incoming notes aligned to musical timing, with performance evaluated through accuracy-based grading and post-level analytics. The system tracks player interactions in real time, providing feedback and scoring that reflects rhythm precision.

The game includes a level editor and item shop, allowing players to customize their gameplay experience and explore new content. Additional configuration options, such as keybindings and audio settings, enhance accessibility and personalization.

A key aspect of development was building responsive UI systems that react dynamically to music. This included implementing animations such as a rhythm-synchronized pulsing main menu, as well as designing and integrating menus, dialogs, and settings systems. Additional work involved debugging and stabilizing systems, particularly within UI flows and the level editor.

Rizumu highlights experience with interactive system design, animation-driven interfaces, and real-time input processing, while demonstrating strong collaboration in a long-term game development project.
